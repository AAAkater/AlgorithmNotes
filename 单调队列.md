# 单调队列
单调队列是一种主要用于解决滑动窗口类问题的数据结构，
即，在长度为n的序列中，求每个长度为m的区间的区间最值

# 不想被单调队列呀QWQ
#### 板子
```cpp
deque<int> Q; // 存储的是编号
for (int i = 0; i < n; ++i)
{
    if (!Q.empty() && i - Q.front() >= m) // 队头出列
        Q.pop_front();
    // 求区间最小值把这里改成>即可
    while (!Q.empty() && V[Q.back()] < V[i])
        Q.pop_back();
    Q.push_back(i); // 新生入队
    if (i+1 >= m)
        cout << V[Q.front()] << " ";
}
```
